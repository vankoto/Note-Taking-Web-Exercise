Below is a description of optional note settings.

Normal use

By simply registering in our web application you can start creating notes right away. The basic options of a note are just creating, deleting and adding text to the note. But of course there are going to be more options, which will be added in the future.

Additional settings

There is one more additional option to creating a note. People sometimes are very busy and do not have the time to write anything at all. The application provides a way of creating a note immediately with sentences which are beforehand established.
